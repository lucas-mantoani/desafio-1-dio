História da Web

Embora o conceito de computadores seja algo antigo, os PCs como conhecemos hoje começaram a existir na década de 70, até evoluirem para dispositivos que podem ser transportados em mochilas facilmente. A internet começou como um sistema de comunição utilizado durante a guerra, a primeira conexão aconteceu em 1969 mas foi 20 anos depois que a WordWideWeb nasceu, criada por Tim Berners-Lee, conhecido como o pai da internet.

O que são Clients?

Os Clients, podem ser representados facilmente como clientes, no modo literal, por exemplo: você entra em um restaurante, pega o menu, faz um pedido para o garçom e ele te retorna com o solicitado, é o mesmo caso com a internet, quando acessamos uma página estamos fazendo uma requisição a um servidor, os navegadores que podem ser nomeados de garçons nos retornam com o solicitado, sem que nós tenhamos que ir até a cozinha e preparar a nossa própria refeição.

O que são Servers?

Os Servers, são computadores que armazenam informações, arquivos, vídeos, músicas, dados, e-mails, e podem ser acessados de forma remota através de uma rede, que pode ser pública ou privada. É ele quem recebe as requisições do cliente e retorna a solitação com html, css, javascript, imagens, vídeos, aúdios e arquivos estáticos.

O que são Linguagens de Programação?

Linguagens de programação são um junto de regras escritas que especificam instruções para gerar programas e aplicações tanto tanto server quanto client-side, cada lado possuí linguaguens especificas, que são escritas de formas diferentes mas que podem ao mesmo resultado desejado.

As 5 principais linguagens de programação são:

Textos

A formatação de texto no HTML pode ser feita de diversas formas. Vamos conhecer as tags mais utilizadas no dia-a-dia:

Paragrafo

A tag p é utilizada para a criação de bloco de textos, pode ser entendida como paragráfos.

Cabeçalhos

As tags h1, h2, h3, h4, h5, h6 são utilizadas para a marcação de cabeçalhos no HTML, cada uma delas possui um nível de importancia sendo H1 a maior e H6 a menor. A tag H1 é utilizada na criação de titulos e as demais em subtitulos

Itálico

A tag I é utilizada para deixar textos em itálico, por exemplo: aqui.

Bold

A tag strong é utilizada para deixar textos mais grossos e dar destaque a palavras como por exemplo aqui.

Mark

A tag mark é utilizada para destacar textos, ela funciona como uma especie de grifador

Sup

A tag sup é utilizada para deixar uma palavra "sobrescrito", frequentemente utilizado para incluir o simbolo de registrado e de direitos.

Sub

A tag sub é utilizada para deixar uma palavra "subescrito", frequentemente utilizado para incluir o simbolo de registrado e de direitos.

Del

A tag Del é utilizada para deixar uma palavra "riscata".
